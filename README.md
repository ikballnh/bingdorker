### Features

- [x] Automation Bing search
- [x] Support dork search
- [x] Bing dorker
- [x] Fast check with multi threading
- [x] 100% Free
#### Requirements
- requests
- CaseInsensitiveDict
- ThreadPoolExecutor
- re
- urllib.parse

```bash
git clone https://github.com/ikballnh/bingdorker
```
```bash
cd bingdorker
```
```bash
pip install -r requirements.txt 
```
```bash
python bing.py
```
